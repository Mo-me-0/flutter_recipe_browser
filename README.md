## Description
A Recipe browser app that allows users to search and explore meal recipes, search for meals, view detailed recipes, ingredients, cooking instructions, and watch YouTube cooking videos using TheMealDB API.
## How to Run
1. Clone the repository
2. Run: flutter pub get
or open in vs code, it automatically add the packages
3. Start emulator or connect device
4. Run: flutter run
## API Endpoints
- GET /categories.php — fetch all categories
- GET /filter.php?c={category} — meals in a category
- GET /lookup.php?i={mealId} — full meal details
- GET /search.php?s={mealName} — search meal by name
